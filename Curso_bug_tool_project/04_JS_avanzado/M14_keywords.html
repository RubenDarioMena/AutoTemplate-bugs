Widget keywords: chips con validación

El widget de palabras clave muestra 'chips' que se añaden con Enter o coma y se borran con Backspace o clic. Valida cantidad exacta, unicidad y que no aparezcan en el cuerpo del bug (kwOverlap).

⚡ JS~10 minIntermedio
1Concepto clave

wireKeywords gestiona los chips. Enter o coma añade, Backspace en input vacío borra el último. La validación verifica: cantidad exacta (kwCount), unicidad (comparar lowercase), y kwOverlap (buscar cada keyword en bodyText del bug). El re-render del widget se hace sin perder el foco del input activo.
